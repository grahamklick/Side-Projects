\# CSV Cleaner



A simple Python project that reads a CSV file, removes duplicate rows and rows with missing values, and writes the cleaned data to a new CSV.



\## Tools Used

\- Python

\- pandas



\## Files

\- `data.csv` - sample raw input data

\- `app.py` - main script

\- `cleaned\_data.csv` - cleaned output file



\## How to Run



Install pandas:



```bash

pip install pandas



Run the script:

python app.py
