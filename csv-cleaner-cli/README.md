\# CSV Cleaner CLI



A command-line Python tool that reads a CSV file, removes duplicate rows and missing values, and writes a cleaned output file.



\## Tools Used

\- Python

\- pandas



\## How to Run



Install pandas:



```bash

pip install pandas



Run the Script:

python app.py data.csv



Or, Point to another file:

python app.py ../csv-cleaner/data.csv
